¿Qué es este sitio?

Éste es un pequeño sitio web frontend alojado y servido con GitHub Pages

Lo voy a utilizar cómo mi carta de presentación cómo desarrollador y a la vez, cómo muestra de mis habilidades con tecnologías front-end en entorno web.

Puesto que es necesario para poder alojarlo cómo sitio web, es uno de los repositorio que tengo en mi cuenta de GitHub

La versión pública está ligeramente minimizada, sin comentar y contiene solo los ficheros que son servidos.

¿Cómo está implementado?

Lo he diseñado e implementado yo y está maquetado con HTML 5 y estilos en cascada CSS 3 compilados con el preprocesador SASS.

Para dar funcionalidad a los botones y otros elementos interactivos, he hecho uso de vanilla JavaScript (sin utilizar frameworks) transpilado con TypeScript.

¿Cuál es el origen?

Este sitio web es la versión frontend y actualizada de mi sitio web full-stack.

La implementación del sitio web empezó sobre 2020 partiendo de desarrollos previos y con motivo de la adquisición de conocimientos en gestión de hosting.

El sitio estuvo disponible a través de los dominios TeCodeo.com y TeCodeo.es y me sirvió para investigar un poquito, el entorno web.

El backend lo escribí en PHP con una arquitectura MVC (Modelo-Vista-Controlador), a modo de una versión muy light de Laravel.

¿Para qué me sirvió?

Dado que estaba buscando trabajo en remoto, TeCodeo.com recibió visitas de todo el mundo.

Lo sé, porque implementé un sistema de registro de visitas y un sistema de telemetría.

Este sistema me permitía saber desde dónde visitaban mi sitio e información sobre el tipo de dispositivo y software utilizado.

Todo ello sin utilizar cookies, y permitiéndome conocer al público interesado para enfocarme en él.

Por otra parte, también me permitió detectar intentos de ejecutar código en la máquina servidor, mediante inyección de scripts en la URL solicitada.
