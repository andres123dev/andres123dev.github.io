Hola, muy buenas, soy Andrés Fernández Burón.

Bienvenido a la página principal de mi sitio web frontend.

Todo el código fuente ha sido escrito por mí desde cero utilizando los lenguajes HTML 5, CSS 3 y vanilla JavaScript.

Además he utilizado TypeScript y el preprocesador SASS.

Recientemente he publicado algunos proyectos propios en mi cuenta de GitHub (andres123dev), que están disponibles cómo repositorio público.

En la sección de Software puedes ver un resumen de las aplicaciones que he publicado.

Cómo puedes ver, hay algunas aplicaciones escritas con Python 3, enfocadas en el análisis de respuestas HTTP.

Por otra parte, hay algunas aplicaciones web.

Dos pizarras front-end para pintar.
Una escrita con vanilla JavaScript y otra utilizando jQuery.
Además de este sitio web.

También una pequeña aplicación web full-stack, que consiste en un CRUD (Create-Read-Update-Delete) conectado a una base de datos de MariaDB.

En el frontend utilizo Bootstrap 5 y realizo las peticiones de forma asíncrona con jQuery y AJAX.
El back-end lo he escrito con el lenguaje PHP.
