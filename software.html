Python 3

My SEO analyzer

My SEO analyzer es un script de Python 3.
Recibe cómo parámetro una URL que sirve el documento HTML a analizar.

Realiza una petición HTTP, hace análisis SEO de la respuesta recibida y genera un reporte en formato PDF.

El repositorio de My SEO analyzer está alojado en mi cuenta de GitHub.
Aquí puedes leer la documentación del proyecto.
Si quieres probarlo puedes descargarlo en formato ZIP.

My URL scraper

My URL scraper es una aplicación de consola de Python 3.
Recibe cómo parámetro la URL a scrapear.

Permite analizar de forma interactiva la respuesta HTTP recibida.
Si la respuesta es un HTML, se pueden realizar búsquedas en el DOM y en el texto del documento.

El repositorio de My URL scraper está alojado en mi cuenta de GitHub.
Aquí puedes leer la documentación del proyecto.

HTML 5, CSS 3, JavaScript & PHP

My CRUD

My CRUD es una aplicación web que permite añadir, modificar, eliminar y listar registros de una tabla de una Base de datos relacional.

El frontend está escrito con HTML 5, CSS 3 y JavaScript, utilizando jQuery y Bootstrap 5.

El backend está escrito con PHP y realiza una conexión con MariaDB utilizando PDO y el patrón singleton.

Las peticiones HTTP se realizan de forma asíncrona mediante AJAX y los formularios se validan en el servidor.

El repositorio de My CRUD está alojado en mi cuenta de GitHub.
Aquí puedes leer la documentación del proyecto.
Si quieres probarla puedes descargarla en formato ZIP.
