HTML5, CSS3, JavaScript & PHP

My CRUD

My CRUD es una aplicación web que permite añadir, modificar, eliminar y listar registros de una tabla de una Base de datos relacional.

El fronted está escrito con HTML 5, CSS 3 y JavaScript, utilizando jQuery y Bootstrap 5.

El backend está escrito con PHP y realiza una conexión con MariaDB utilizando PDO y el patrón singleton.

Los formularios se validan en el servidor y las peticiones se realizan de forma asíncrona mediante AJAX.

El repositorio de My CRUD está alojado en mi cuenta de GitHub.
