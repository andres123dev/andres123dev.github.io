Python 3

My SEO analyzer

My SEO analyzer sirve para realizar el análisis SEO de la respuesta a una petición HTTP.

Es un script escrito con Python 3.

Recibe cómo parámetro la URL del documento HTML a analizar.

Realiza la petición HTTP, hace un análisis SEO de la respuesta recibida y genera un reporte en formato PDF.

El repositorio y la documentación de My SEO analyzer están alojados en mi cuenta de GitHub.

Si quieres probar el script, puedes descargarlo cómo ZIP directamente.

My URL scraper

My URL scraper sirve para scrapear de forma interactiva una URL.

Es una aplicación de consola escrita con Python 3.

Recibe cómo parámetro la URL a scrapear.

Permite analizar de manera interactiva la respuesta HTTP recibida.

Si la respuesta es un HTML, se pueden realizar búsquedas en el DOM y en el texto del documento.

El repositorio y la documentación de My URL scraper están alojados en mi cuenta de GitHub.

Si quieres probar la aplicación, puedes descargarla cómo ZIP directamente.

HTML 5, CSS 3, JavaScript & PHP

My CRUD

My CRUD es una aplicación web que permite añadir, modificar, eliminar y listar registros de una tabla de una Base de datos relacional.

El frontend está escrito con HTML 5, CSS 3 y JavaScript, utilizando jQuery y Bootstrap 5.

El backend está escrito con PHP y realiza una conexión con MariaDB utilizando PDO y el patrón singleton.

Las peticiones HTTP se realizan de forma asíncrona mediante AJAX y los formularios se validan en el servidor.

El repositorio y la documentación de My CRUD están alojados en mi cuenta de GitHub.

Si quieres probar la aplicación, puedes descargarla cómo ZIP directamente.
