My CRUD

Es una aplicación web que permite añadir, modificar, eliminar y listar registros de una tabla de una Base de datos relacional.

Es un CRUD (Create-Read-Update-Delete) web escrito en 2023 en formato SPA (Single Page Application).

El frontend está escrito con HTML 5, CSS 3 y JavaScript, utilizando jQuery y Bootstrap 5.

El backend está escrito con PHP y realiza una conexión con MariaDB utilizando PDO y el patrón singleton.

Las peticiones HTTP se realizan de forma asíncrona mediante AJAX y los formularios se validan en el servidor.

Gestión de concesionarios

Es una aplicación de escritorio con interfaz gráfica y acceso a datos.

Es una aplicación escrita en 2019 junto con una compañera, cómo proyecto final para las asignaturas de Interfaces gráficas y Acceso a datos del CFGS Desarrollo de Aplicaciones Multiplataforma.

Está escrita utilizando el lenguaje Java SE y la librería gráfica Swing.

Tiene conexión a una Base de datos de MySQL que consta de cuatro tablas relacionadas (Coche, Empleado, Concesionario, Cliente) y permite:

  • Conectar con el Sistema Gestor de Base de Datos
  • Añadir, modificar y eliminar registros
  • Realizar consultas con filtros
  • Importar registros desde ficheros TXT, CSV, XML y HTML
  • Exportar registros a ficheros TXT, CSV y XML
